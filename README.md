# Лабораторна робота №1

**Дисципліна:** Web-дизайн  
**Тема:** Розробка статичного інтерфейсу Web-додатка  

## Інформація про розробника
* **Студент:** Кравець Андрій  
* **Група:** КВ-32  

## Завдання
Розробити статичні Web-сторінки додатку та перевірити коректне їх відображення на екранах різного розміру, використовуючи HTML5, CSS3 та фреймворк Twitter Bootstrap. 

**Обрана тематика додатка:** Калькулятор (режим звичайний, для програміста).

**Перелік розроблених сторінок:**
1. `index.html` - Робоча сторінка додатка (Калькулятор).
2. `profile.html` - Профіль користувача.
3. `about.html` - Про додаток.
4. `login.html` - Вхід до сайту.
5. `register.html` - Реєстрація користувача.
